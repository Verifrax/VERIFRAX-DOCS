# VERIFRAX-DOCS

Documentation for the VERIFRAX protocol stack.

## Scope
- Public reading surface
- Integration and usage guidance
- Determinism + finality model

## Surfaces
- VERIFRAX (authoring + normative source material): https://github.com/Verifrax/VERIFRAX
- SPEC (derived specification projection): https://github.com/Verifrax/VERIFRAX-SPEC
- PROFILES (overlays): https://github.com/Verifrax/VERIFRAX-PROFILES
- SAMPLES (runnable): https://github.com/Verifrax/VERIFRAX-SAMPLES
- verify (standalone): https://github.com/Verifrax/VERIFRAX-verify
- AUCTORISEAL (authority): https://github.com/Verifrax/AUCTORISEAL
- CORPIFORM (governed execution): https://github.com/Verifrax/CORPIFORM

## Authority boundary
- VERIFRAX authors the active normative source material.
- VERIFRAX-SPEC publishes derived specification artifacts from VERIFRAX.
- Derived specification artifacts are not upstream repository authority.

## Policy
- Keep documentation aligned with live repository truth.
- Do not describe unpublished surfaces as public packages.
- Do not describe derived publication surfaces as upstream repository authority.
